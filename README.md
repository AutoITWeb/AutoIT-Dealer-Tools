# Biltorvet Dealer Tools

Adds Vehicle search, Vehicle list and Vehicle Detail Page.

**Features**

* Integration to AutoDesktop
* Search in the dealers cars
* Show car-data
* Send leads to AutoDesktop


**Credits**

Developed by [Biltorvet A/S](https://biltovet.as)
